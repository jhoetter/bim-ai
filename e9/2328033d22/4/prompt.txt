T3 — R3-04: OutlinePass selection + factory cleanup

  ▎ Start with: git checkout -b r3-04-outline-selection

  You are working on the 3D viewport of a BIM (architectural CAD) web app.
  Primary file: packages/web/src/Viewport.tsx (~2100 lines).

  CONTEXT — CURRENT STATE
  =======================
  Selection is currently implemented by passing a `selectedId?: string`
  parameter into every factory function (makeWallMesh, makeWallWithOpenings,
  makeDoorMesh, makeWindowMesh, makeStairVolumeMesh). Each factory does a
  ternary like:
    color: wall.id === selectedId
      ? (paint?.selection.selectedColor ?? '#fb923c')
      : categoryColorOr(paint, 'wall')

  This means the entire scene is rebuilt (the full useEffect re-runs) whenever
  selectedId changes. The rebuild dependency array includes `selectedId` at
  line ~1989.

  The EffectComposer chain is: RenderPass → SSAOPass → OutputPass
  (lines ~1385–1396).

  The factory functions with selectedId today:
    makeStairVolumeMesh(stair, elementsById, paint, selectedId?)  ~line 635
    makeWallMesh(wall, elevM, paint, selectedId?)                 ~line 719
    makeWallWithOpenings(wall, elevM, doors, wins, paint, selectedId, evaluator) ~line 750
    makeDoorMesh(door, wall, elevM, paint, sid?)                  ~line 833
    makeWindowMesh(win, wall, elevM, paint, sid?)                 ~line 884

  TASK
  ====
  This task has three parts:

  ── PART A: OutlinePass ─────────────────────────────────────────────────────

  1. Add import at top of file:
       import { OutlinePass } from 'three/addons/postprocessing/OutlinePass.js';

  2. Add a ref:
       const outlinePassRef = useRef<OutlinePass | null>(null);

  3. In the renderer/scene init useEffect, insert OutlinePass between SSAOPass
     and OutputPass (currently lines ~1395–1396):
       // BEFORE (current):
       composer.addPass(ssao);
       composer.addPass(new OutputPass());

       // AFTER:
       composer.addPass(ssao);
       const outlinePass = new OutlinePass(
         new THREE.Vector2(host.clientWidth || 1, host.clientHeight || 1),
         scene,
         camera,
       );
       outlinePass.edgeStrength  = 3.0;
       outlinePass.edgeGlow      = 0.3;
       outlinePass.edgeThickness = 1.5;
       // colours read from paint (resolved before this block)
       outlinePass.visibleEdgeColor.set(paint.selection.selectedColor);
       outlinePass.hiddenEdgeColor.set(paint.selection.selectedColor);
       composer.addPass(outlinePass);
       outlinePassRef.current = outlinePass;
       composer.addPass(new OutputPass());

  4. After the scene-rebuild useEffect re-builds the scene (toward the end,
     after root.traverse for shadows), update the outline pass to reflect
     the current selection. Add a ref for the bimPickId→Object3D map:
       const bimPickMapRef = useRef<Map<string, THREE.Object3D>>(new Map());

     After all scene objects are added to root:
       const pickMap = new Map<string, THREE.Object3D>();
       root.children.forEach(obj => {
         const id = obj.userData.bimPickId as string | undefined;
         if (id) pickMap.set(id, obj);
       });
       bimPickMapRef.current = pickMap;

       // Reapply current selection to outline pass
       const op = outlinePassRef.current;
       if (op) {
         const sel = selectedId ? pickMap.get(selectedId) : undefined;
         op.selectedObjects = sel ? [sel] : [];
         op.visibleEdgeColor.set(paint.selection.selectedColor);
         op.hiddenEdgeColor.set(paint.selection.selectedColor);
       }

  5. Add a separate useEffect that fires ONLY when selectedId changes (not
     full scene rebuild):
       useEffect(() => {
         const op = outlinePassRef.current;
         if (!op) return;
         const sel = selectedId
           ? bimPickMapRef.current.get(selectedId)
           : undefined;
         op.selectedObjects = sel ? [sel] : [];
       }, [selectedId]);

  6. Remove `selectedId` from the scene-rebuild useEffect dependency array
     (find the array at line ~1987 and remove the `selectedId,` entry).
     The scene no longer needs to rebuild when selection changes.

  ── PART B: remove selectedId from factory functions ───────────────────────

  Remove the `selectedId` / `sid` parameter from these five functions AND
  remove the selection ternary color logic inside each:

  makeStairVolumeMesh: remove `selectedId?: string` param; change:
    const stairColor = stair.id === selectedId ? ... : categoryColorOr(paint,'stair');
    const mat = new THREE.MeshStandardMaterial({ color: stairColor });
    to:
    const mat = new THREE.MeshStandardMaterial({
      color: categoryColorOr(paint, 'stair'),
      roughness: paint?.categories.stair.roughness ?? 0.85,
    });

  makeWallMesh: remove `selectedId?: string` param; change:
    color: wall.id === selectedId ? ... : categoryColorOr(paint, 'wall'),
    to:
    color: categoryColorOr(paint, 'wall'),
    and add:
    metalness: paint?.categories.wall.metalness ?? 0.0,

  makeWallWithOpenings: remove `selectedId: string | undefined` param;
    update the wallMat construction the same way (no ternary, add metalness);
    update the fallback call: makeWallMesh(wall, elevM, paint)  ← no selectedId.

  makeDoorMesh: remove `sid?: string` param; change:
    const doorColor = door.id === sid ? ... : categoryColorOr(paint, 'door');
    const frameMat = new THREE.MeshStandardMaterial({ color: doorColor });
    const panelMat = new THREE.MeshStandardMaterial({ color: doorColor, roughness: 0.75 });
    to:
    const frameColor = categoryColorOr(paint, 'door');
    const frameMat = new THREE.MeshStandardMaterial({
      color: frameColor,
      roughness: paint?.categories.door.roughness ?? 0.70,
      metalness: paint?.categories.door.metalness ?? 0.0,
    });
    const panelMat = new THREE.MeshStandardMaterial({
      color: frameColor,
      roughness: paint?.categories.door.roughness ?? 0.70,
    });

  makeWindowMesh: remove `sid?: string` param; change:
    const frameColor = win.id === sid ? ... : categoryColorOr(paint, 'window');
    const frameMat = new THREE.MeshStandardMaterial({ color: frameColor });
    to:
    const frameMat = new THREE.MeshStandardMaterial({
      color: categoryColorOr(paint, 'window'),
      roughness: paint?.categories.window.roughness ?? 0.60,
      metalness: paint?.categories.window.metalness ?? 0.05,
    });

  Also fix makeRoofMassMesh (~line 619): change hardcoded `metalness: 0.04`
  to `metalness: paint?.categories.roof.metalness ?? 0.0`.
