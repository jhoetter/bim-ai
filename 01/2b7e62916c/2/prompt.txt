  CONTEXT
  =======
  CameraRig (cameraRig.ts) has a `snapshot()` method returning
  { position, target, up, radius, azimuth, elevation }.
  The viewport uses a single THREE.PerspectiveCamera (fov=55) stored in
  cameraRef. The EffectComposer RenderPass is built with this camera.
  The composer and all passes that receive `camera` must be updated when
  the active camera changes.

  TASK — cameraRig.ts
  ===================
  Add a method to CameraRig that computes the matching OrthographicCamera
  frustum for the current rig state:

    orthoFrustum(aspect: number): {
      left: number; right: number; top: number; bottom: number;
      near: number; far: number;
    } {
      const halfH = this.state.radius * Math.tan(
        THREE.MathUtils.degToRad(27.5)  // half of default 55° fov
      );
      return {
        left:   -halfH * aspect,
        right:   halfH * aspect,
        top:     halfH,
        bottom: -halfH,
        near:   0.05,
        far:    500,
      };
    }
