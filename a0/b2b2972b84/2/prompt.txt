  TASK
  ====
  1. Add Sky env map infrastructure.

     Import at top of file (after the existing Three.js addons imports):
       import { Sky } from 'three/addons/objects/Sky.js';

     Write a top-level helper function (NOT inside a component):
       function buildSkyEnvMap(
         renderer: THREE.WebGLRenderer,
         azimuthDeg: number,
         elevationDeg: number,
       ): THREE.Texture

     Inside buildSkyEnvMap:
     - Create a Sky instance; set sky.scale.setScalar(450000)
     - Configure uniforms: turbidity=3, rayleigh=0.5, mieCoefficient=0.005,
       mieDirectionalG=0.8
     - Compute sun position:
         const phi   = THREE.MathUtils.degToRad(90 - elevationDeg);
         const theta = THREE.MathUtils.degToRad(azimuthDeg);
         sky.material.uniforms['sunPosition'].value
           .setFromSphericalCoords(1, phi, theta);
     - Render to PMREM:
         const pmrem = new THREE.PMREMGenerator(renderer);
         pmrem.compileEquirectangularShader();
         const tempScene = new THREE.Scene();
         tempScene.add(sky);
         const envTexture = pmrem.fromScene(tempScene).texture;
         pmrem.dispose();
         (sky.material as THREE.ShaderMaterial).dispose();
         sky.geometry.dispose();
         return envTexture;

  2. Add an envMapRef to store the current env texture:
       const envMapRef = useRef<THREE.Texture | null>(null);

  3. In the renderer/scene init useEffect (the one starting ~line 1320 that
     runs once on mount), after the scene and renderer are created and paint
     is resolved but before the animation loop starts, build and apply the
     env map:
       const envMap = buildSkyEnvMap(
         renderer,
         paint.lighting.sun.azimuthDeg,
         paint.lighting.sun.elevationDeg,
       );
       scene.environment = envMap;
       envMapRef.current = envMap;

  4. Rebuild the env map when the theme changes. In the scene-rebuild useEffect
     (the one with the big dependency array ~line 1987), after the root group
     is populated (near the end, before return), add:
       const rnd = rendererRef.current;
       const sc  = sceneRef.current;
       if (rnd && sc) {
         envMapRef.current?.dispose();
         const newEnv = buildSkyEnvMap(
           rnd,
           paint.lighting.sun.azimuthDeg,
           paint.lighting.sun.elevationDeg,
         );
         sc.environment = newEnv;
         envMapRef.current = newEnv;
       }

  5. Apply envMapIntensity to the glazing material inside makeWindowMesh.
     Find the existing glazing MeshStandardMaterial and add:
       envMapIntensity: 1.2,
     (Already has roughness: 0.05 — leave that unchanged.)

  6. Cleanup: in the renderer cleanup / unmount teardown (wherever the renderer
     and composer are disposed), also dispose envMapRef.current if set.

  7. DO NOT touch packages/web/src/viewport/materials.ts.
     DO NOT change the EffectComposer pass chain (that's handled by a separate
     parallel task). Only add the Sky env map infrastructure.
