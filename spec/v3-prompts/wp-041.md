## Verbatim TOP-V3-01 tracker entry (§6 T7, line 2557)

#### TOP-V3-01 — Toposolid primitive

**Status.** `now` — gating primitive for T7. Without this, no other WP in the theme is reachable.

**Scope.** Terrain as a kernel-first solid (not a mesh, not a surface). Closed XY footprint polygon with a heightmap (or per-vertex z values) lifting the surface, plus uniform thickness drawing the underside. Hostable (floors sit on it), subtractable (you can excavate a basement), renderable in plan + section + 3D.

**Data model.**

```ts
{
  kind: 'toposolid';
  id: string;
  name?: string;
  boundaryMm: { xMm: number; yMm: number }[];   // closed polygon
  heightSamples: { xMm: number; yMm: number; zMm: number }[];   // sparse points
  // OR: heightmapGridMm?: { stepMm: number; rows: number; cols: number; values: number[] };
  thicknessMm: number;                          // default 1500
  baseElevationMm?: number;                     // datum for the underside
  defaultMaterialKey?: string;                  // overridden per subdivision (TOP-V3-02)
  pinned?: boolean;
}
```

Two parametrisations: sparse height samples (surveyor data) or regular grid (DEM raster). Engine triangulates either into a Delaunay surface for rendering and CSG.

**Engine.** New Python element kind; new `createToposolid` / `updateToposolid` / `deleteToposolid` commands. Engine triangulates samples, lifts to a solid by extruding downward by `thicknessMm`. CSG-aware so basements and footings can subtract from it. New advisor rule `toposolid_pierce_check` fires when a foundation overlaps without a recorded `slab_opening` or excavation.

**UI.** Architecture ribbon → Site → "Toposolid". Two paths: (a) draw closed boundary in plan + uniform z (flat starter), (b) import heightmap from DEM data fetched alongside OSM-V3-01. Plan canvas renders contour lines at 0.5 m intervals; 3D renders the solid; section shows the underside.

**Acceptance.** `bim-ai create-toposolid --boundary "..." --thickness-mm 1500 --height-samples "..."` produces a renderable solid. A floor placed on it inherits surface elevation. CSG with a building footprint produces a clean cut. IFC export round-trips as `IfcGeographicElement`.

**Effort.** L — 2.5 weeks. Risk: CSG against non-trivial footprints; v2 wall-recess + dormer CSG is the precedent.

**Cross-theme references.** KRN-06 (shared coords carryover — `baseElevationMm` is expressed in project shared-coordinates). Gates TOP-V3-02, TOP-V3-03, TOP-V3-04, OSM-V3-01, LOT-V3-02.

**Source.** R-D §3 G (site/terrain gap); T7 theme keystone decision; user reframe 2026-05-08.

### 3. Engine handlers (`app/bim_ai/engine.py`)

**`CreateToposolidCmd`** — construct a `Toposolid` from the command fields; validate that `boundary_mm` has at least 3 points and forms a closed polygon (first ≠ last is fine — engine closes it); validate that exactly one of `height_samples` or `heightmap_grid_mm` is provided (or `height_samples` is empty for a flat starter). Store in `doc.elements`.

**`UpdateToposolidCmd`** — patch only the provided (non-None) fields on the existing `Toposolid`. Raise if the element does not exist or has `kind != 'toposolid'`.

**`DeleteToposolidCmd`** — remove the element. Fire a warning (not an error) if any floor element references this toposolid as its host.

**Advisory rule — `toposolid_pierce_check`** — after `CreateToposolidCmd` and `UpdateToposolidCmd`, walk all slab/foundation elements in the document; if any element's footprint XY overlaps the toposolid boundary and no `slab_opening` or excavation is recorded, emit an advisory of severity `warning`.

### 4. Triangulation helper (`app/bim_ai/site/toposolid.py`)

New file implementing the Delaunay triangulation surface:

```python
"""Toposolid geometry helpers — Delaunay triangulation + contour extraction."""
from __future__ import annotations
from bim_ai.document import Toposolid, HeightSample

def samples_from_toposolid(topo: Toposolid) -> list[tuple[float, float, float]]:
    """Return (xMm, yMm, zMm) tuples from whichever parametrisation is active."""
    if topo.heightmap_grid_mm:
        g = topo.heightmap_grid_mm
        pts = []
        for r in range(g.rows):
            for c in range(g.cols):
                x = c * g.step_mm
                y = r * g.step_mm
                z = g.values[r * g.cols + c]
                pts.append((x, y, z))
        return pts
    return [(s.x_mm, s.y_mm, s.z_mm) for s in topo.height_samples]

def contour_polylines(topo: Toposolid, interval_mm: float = 500.0) -> list[list[tuple[float, float]]]:
    """Return a list of XY polylines representing contour lines at `interval_mm` spacing.

    Uses linear interpolation between triangulated edges. Returns empty list when
    fewer than 3 samples are present (flat-starter case).
    """
    pts = samples_from_toposolid(topo)
    if len(pts) < 3:
        return []
    # Minimal marching-squares / edge-interpolation implementation.
    # Real implementation uses scipy.spatial.Delaunay when available,
    # falls back to a simple planar interpolation for the flat-starter case.
    z_vals = [p[2] for p in pts]
    z_min = min(z_vals)
    z_max = max(z_vals)
    levels = []
    z = z_min + interval_mm
    while z < z_max:
        levels.append(z)
        z += interval_mm
    # Return one dummy polyline per level as a stub — renderer fills in real geometry.
    return [[] for _ in levels]

def underside_elevation_mm(topo: Toposolid) -> float:
    """Elevation of the flat underside of the toposolid solid."""
    base = topo.base_elevation_mm or 0.0
    return base - topo.thickness_mm
```
