## Background
TOP-V3-03 is the painterly authoring UX for toposolid subdivisions: pick a toposolid → "Add subdivision" → sketch polygon on canvas → assign finish category from a palette drawer → commit. The feel is "decorating terrain, not editing a CAD object."

This WP is **frontend-only**. The data model and engine commands live in WP-064. This WP adds:
1. A `ToposolidSubdivisionTool` frontend state machine
2. A finish-category palette drawer component
3. Hotkey `TS` to activate
4. Wire the tool to dispatch `CreateToposolidSubdivisionCmd`

## Implementation tasks

### 1. Tool state in `toolPrefsStore.ts` or dedicated slice

Add to `packages/web/src/tools/toolPrefsStore.ts`:
```typescript
// Toposolid subdivision tool state
subdivisionDraft: {
  hostToposolidId: string | null;
  boundaryPts: { xMm: number; yMm: number }[];
  finishCategory: 'paving' | 'lawn' | 'road' | 'planting' | 'other';
} | null;
setSubdivisionDraft: (draft: typeof state.subdivisionDraft) => void;
clearSubdivisionDraft: () => void;
```

### 2. Finish-category palette drawer (`packages/web/src/workspace/SubdivisionPalette.tsx`)

```tsx
// SubdivisionPalette — hovering drawer with 5 finish categories
// Appears when a toposolid is selected + "Add subdivision" is active
// Uses var(--color-surface-2), var(--radius-md), var(--shadow-modal)
// Zero hex literals — CSS vars only

const CATEGORIES = [
  { id: 'paving', label: 'Paving', icon: '▦' },
  { id: 'lawn', label: 'Lawn', icon: '▩' },
  { id: 'road', label: 'Road', icon: '▬' },
  { id: 'planting', label: 'Planting', icon: '✿' },
  { id: 'other', label: 'Other', icon: '◻' },
] as const;
```

Props: `{ activeCategory, onSelect, onCancel }`. Render as a 5-button row, each button has the icon + label. Active category gets `var(--color-accent)` background. Esc calls `onCancel`.

### 3. Subdivision sketch mode in `PlanCanvas.tsx`

When `subdivisionDraft` is non-null and `activeTool === 'toposolid_subdivision'`:
- Cursor changes to crosshair
- Click → add vertex to `boundaryPts`
- Double-click → close polygon + emit `CreateToposolidSubdivisionCmd` with collected boundary + selected finishCategory
- Esc → clear draft

Show live preview: a dashed polygon line connecting accumulated pts as the user clicks.

### 4. Hotkey `TS`

In the keyboard handler (wherever tool hotkeys are registered), add `TS` (T then S sequence within 500ms) to activate `toposolid_subdivision` tool. Must only activate when a toposolid element is selected.

### 5. Tests (`packages/web/src/workspace/SubdivisionPalette.test.tsx`)

Write at least 6 tests:
- Renders 5 category buttons
- Active category highlighted
- Click category calls onSelect
- Esc calls onCancel
- Zero hex color literals in component (regex check on rendered HTML)
- All buttons accessible (aria-label present)
