## Implementation tasks

### 1. Empty-state hint component (`packages/web/src/workspace/EmptyStateHint.tsx`)

Create a new component:

```tsx
import { useEffect, useState } from 'react';

export function EmptyStateHint() {
  const [visible, setVisible] = useState(true);

  useEffect(() => {
    // Fade out after 6 s
    const hideTimer = setTimeout(() => setVisible(false), 6000);
    // Re-show after 60 s idle (reset on any mouse move)
    let idleTimer: ReturnType<typeof setTimeout>;
    const resetIdle = () => {
      setVisible(false);
      clearTimeout(idleTimer);
      idleTimer = setTimeout(() => setVisible(true), 60_000);
    };
    window.addEventListener('mousemove', resetIdle);
    return () => {
      clearTimeout(hideTimer);
      clearTimeout(idleTimer);
      window.removeEventListener('mousemove', resetIdle);
    };
  }, []);

  return (
    <div
      style={{
        position: 'absolute',
        inset: 0,
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        pointerEvents: 'none',
        opacity: visible ? 1 : 0,
        transition: 'opacity 320ms var(--ease-paper)',
      }}
    >
      <p
        style={{
          fontFamily: 'var(--font-sans)',
          fontSize: 'var(--text-display)',
          color: 'var(--color-muted-foreground)',
          textAlign: 'center',
          userSelect: 'none',
        }}
      >
        Select an element, or press <kbd>W</kbd> to draw a wall.
      </p>
    </div>
  );
}
```

No hex literals. All colours and typography via `var(--*)` tokens.

### 2. Mount in the canvas area

In the main canvas wrapper component (wherever the plan canvas is rendered), mount `<EmptyStateHint />` only when:

- No elements are selected (`selectedIds.length === 0`)
- No drawing tool is active

```tsx
const selectedIds = useStore(s => s.selectedIds);
const activeTool = useStore(s => s.activeTool);
const isEmpty = selectedIds.length === 0 && !activeTool;

// In JSX:
{isEmpty && <EmptyStateHint />}
```

### 3. Canvas extends to right edge

When the right rail is hidden (CHR-V3-06 removes it from DOM when nothing selected), the canvas should naturally extend to fill the available width. Verify the canvas container uses `flex: 1` or equivalent so it fills the remaining space when the inspector rail is absent.

If the canvas has a hard-coded right margin or fixed width that doesn't adapt, fix it to be responsive to the rail's presence/absence.

### 4. `<kbd>` styling

The `<kbd>W</kbd>` in the hint should look like a keyboard key: `--surface-2` background, `--radius-sm`, `--color-border` 1 px border, `--text-xs` monospace. Add a small CSS class for kbd within the workspace scope.
