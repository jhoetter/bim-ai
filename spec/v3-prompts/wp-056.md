## Verbatim EDT-V3-09 tracker entry

**Scope.** In stair-sketch mode (authoring_mode=`by_sketch`), the user sees boundary + N tread lines auto-distributed per rise-run rule (`tread ≥ 260 mm × riser ≤ 190 mm` advisory). Dragging one tread line **auto-rebalances** the others — pulling wider narrows adjacent ones proportionally; total run stays constant. `manualOverride: true` flags treads that skip rebalancing. `Shift+drag` sets override; `Cmd+R` resets all overrides.

**Engine.** New `app/bim_ai/stair/autobalance.py`: `rebalance_treads(tread_lines, moved_index, new_position_mm, total_run_mm) → list[StairTreadLine]`. Closed-form for straight stairs; proportional redistribution among unlocked treads.

**UI.** Drag tread → adjacent treads slide live at 60 fps. Helper-dim chips (EDT-V3-06) on each tread show the new width. `Shift+drag` sets `manualOverride`; `Cmd+R` resets.

**Acceptance.** 14-tread stair → treads auto-distribute. Drag tread #5 wider → #4 and #6 narrow to compensate. `Shift+drag` tread #3 locks it; subsequent drags rebalance only unlocked treads.

### 2. New autobalance module (`app/bim_ai/stair/autobalance.py`)

Create `app/bim_ai/stair/__init__.py` (empty) and `app/bim_ai/stair/autobalance.py`:

```python
"""EDT-V3-09: tread auto-balance for by_sketch stairs."""
from __future__ import annotations
from app.bim_ai.elements import StairTreadLine, Vec2Mm
import math


def rebalance_treads(
    tread_lines: list[StairTreadLine],
    moved_index: int,
    new_from_mm: Vec2Mm,
    total_run_mm: float,
) -> list[StairTreadLine]:
    """Rebalance tread lines after moving one, keeping total run constant.

    Locked treads (manualOverride=True) keep their position. Unlocked treads
    (excluding the moved one) redistribute proportionally.
    """
    if not tread_lines or moved_index < 0 or moved_index >= len(tread_lines):
        return tread_lines

    n = len(tread_lines)
    result = [t.model_copy() for t in tread_lines]

    # Place the moved tread at the new position
    moved = result[moved_index]
    tread_width = _tread_width(moved)
    result[moved_index] = moved.model_copy(
        update={"from_mm": new_from_mm, "to_mm": Vec2Mm(x_mm=new_from_mm.x_mm + tread_width, y_mm=new_from_mm.y_mm)}
    )

    # Collect unlocked treads excluding the moved one
    unlocked_indices = [
        i for i in range(n)
        if i != moved_index and not result[i].manual_override
    ]
    if not unlocked_indices:
        return result

    # Compute remaining run for unlocked treads
    locked_run = sum(
        _tread_width(result[i])
        for i in range(n)
        if i != moved_index and result[i].manual_override
    )
    moved_run = _tread_width(result[moved_index])
    remaining_run = total_run_mm - locked_run - moved_run
    if remaining_run <= 0 or len(unlocked_indices) == 0:
        return result

    # Distribute remaining_run equally among unlocked treads
    new_width = remaining_run / len(unlocked_indices)
    for i in unlocked_indices:
        t = result[i]
        result[i] = t.model_copy(
            update={"to_mm": Vec2Mm(x_mm=t.from_mm.x_mm + new_width, y_mm=t.from_mm.y_mm)}
        )

    return result


def auto_distribute_treads(
    boundary_mm: list[Vec2Mm],
    total_rise_mm: float,
    tread_mm: float = 275.0,
    riser_mm: float = 175.0,
) -> list[StairTreadLine]:
    """Generate initial tread distribution from stair boundary."""
    if len(boundary_mm) < 2:
        return []
    # Compute run length from boundary
    start = boundary_mm[0]
    end = boundary_mm[-1]
    total_run = math.sqrt((end.x_mm - start.x_mm) ** 2 + (end.y_mm - start.y_mm) ** 2)
    n_treads = max(1, round(total_run / tread_mm))
    w = total_run / n_treads
    lines = []
    for i in range(n_treads):
        x0 = start.x_mm + i * w
        x1 = start.x_mm + (i + 1) * w
        lines.append(StairTreadLine(
            fromMm=Vec2Mm(x_mm=x0, y_mm=start.y_mm),
            toMm=Vec2Mm(x_mm=x1, y_mm=start.y_mm),
            riserHeightMm=total_rise_mm / (n_treads + 1),
        ))
    return lines


def _tread_width(t: StairTreadLine) -> float:
    return abs(t.to_mm.x_mm - t.from_mm.x_mm)
```

case UpdateStairTreadsCmd():
    stair = els.get(cmd.id)
