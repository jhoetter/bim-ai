### 2. Recency store (`packages/web/src/cmdPalette/paletteRecencyStore.ts`)

Zustand slice:
- `invocations: Record<string, { ts: number; count: number }>` (last 100).
- `recordInvocation(id: string)` — upsert; trim to 100; persist to `localStorage`.
- `getRecencyScore(id: string, now: number): number` — exponential decay over 14 days: `count * exp(-decay * (now - ts) / (14 * 86400 * 1000))`.
- Hydrates from `localStorage` on init.

### 4. Keybinding + mount (`packages/web/src/workspace/Workspace.tsx`)

Add a `useEffect` in `Workspace.tsx`:

```ts
useEffect(() => {
  const handler = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.key === 'k') {
      e.preventDefault();
      openPalette();
    }
  };
  window.addEventListener('keydown', handler);
  return () => window.removeEventListener('keydown', handler);
}, [openPalette]);
```

Mount `<CommandPalette>` at the root of `Workspace.tsx` (outside the canvas layers so it overlays everything).

## Implementation rules

- **OS-aware keybinding.** Use `e.metaKey` for Mac, `e.ctrlKey` for Win/Linux. `navigator.platform` or `navigator.userAgent` to distinguish. Both must work.
- **No hex literals.** All colours in TSX/CSS use CSS token variables.
- **`--radius-canvas: 0` exception.** The palette is a floating modal, not a canvas — use `--radius-card` (if it exists) or `border-radius: 4px`. Do NOT use `--radius-canvas: 0` here.
- **fuzzysort.** If not in `package.json`, add it: `pnpm add fuzzysort`. Do not vendor it.
- **Registry is append-only at runtime.** `registerCommand` pushes; nothing removes at runtime. Clearing is only in tests.
