Add commands in `app/bim_ai/commands.py`:
```python
class ImportImageUnderlayCmd(BaseModel):
    type: Literal["import_image_underlay"] = "import_image_underlay"
```

Add all five to `Command` union.

### 2. Engine handlers (`app/bim_ai/engine.py`)

- `ImportImageUnderlay`: validate format (src must start with `data:image/png`, `data:image/jpeg`, or `data:application/pdf`); validate src size ≤ 50 MB; create element.
- `MoveImageUnderlay`: update `rect_mm` (position only, preserve size).
- `ScaleImageUnderlay`: update `rect_mm` width/height.
- `RotateImageUnderlay`: update `rotation_deg`.
- `DeleteImageUnderlay`: remove element.
