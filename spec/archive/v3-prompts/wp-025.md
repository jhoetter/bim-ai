## Verbatim KRN-V3-08 tracker entry (§6 T1, line 715)

**KRN-V3-08 — Wall sweeps & reveals**

_Source: R-D §3 G2. Severity Medium. Cornice, water-table, plinth — first-class wall children in Revit._

**Status.** `next`.
**Scope.** Horizontal profile run along a wall (sweep — additive) or inverse cut (reveal). Reuses the `EdgeProfileRun` infra shipped in KRN-V3-03 — wall sweeps are just edge-profile-runs hosted to a wall edge instead of a roof edge. Auto-mitred at corners.
**Data model.** Reuses `EdgeProfileRun` shape with `hostElementId` pointing at a wall and `hostEdge: 'top' | 'bottom' | { startMm; endMm }`. Adds `mode: 'sweep' | 'reveal'`.
**Engine.** Sweep: solid added to wall via boolean union. Reveal: solid subtracted from wall via boolean difference. Auto-mitering at wall-wall corners. Profile families: cornice, water-table, plinth, baseboard, chair-rail, picture-rail.
**Acceptance.** Apply a cornice sweep at the top edge of every exterior wall; cornices auto-mitre at corners; in elevation the cornice reads as a continuous trim band.
**Effort.** M — 1 week.

### 2. Extend `EdgeProfileRun` for wall hosting (if not already general)

If `host_edge` is currently roof-specific (e.g. `'eave' | 'rake'`), extend it to also accept wall edges:

```python
host_edge: (
    Literal["top", "bottom", "eave", "rake"]

```

Where `WallEdgeSpan` is:
```python
class WallEdgeSpan(BaseModel):
    model_config = ConfigDict(populate_by_name=True, extra="ignore")
    start_mm: float = Field(alias="startMm")   # along wall length
    end_mm: float = Field(alias="endMm")
```

If `mode: 'sweep' | 'reveal'` is not already on `EdgeProfileRun`, add it:
```python

```

### 3. Engine — wall-hosted sweep/reveal logic (`engine.py`)

In the `EdgeProfileRun` handler (or a new helper `_apply_wall_edge_profile`), add a branch when `host_element_id` references a `WallElem`:

**Sweep (`mode='sweep'`):**
- Extrude the profile solid along the wall's length at the specified host edge elevation.
- For `host_edge='top'`: place the profile run flush with the wall's top.
- For `host_edge='bottom'`: place flush with the wall's base.
- For `WallEdgeSpan`: start/end clamp to wall length.
- Result is stored as a child of the wall (or as a top-level element with `parent_id`).

**Reveal (`mode='reveal'`):**
- Same placement, but the solid is subtracted from the wall instead of added.
- Produce a `WallRevealCut` record on the wall element listing the cut geometry.

**Auto-mitering:**
- When two adjacent walls share a corner, sweeps at the same `host_edge` and same profile extend to form a mitered joint. Implement a pure function:

```python
def _compute_wall_sweep_mitre_angle(
    wall_a_angle_deg: float,
    wall_b_angle_deg: float,
) -> float:
    """Return the cut angle for a mitre joint between two wall sweeps."""
    return (wall_a_angle_deg + wall_b_angle_deg) / 2.0
```
