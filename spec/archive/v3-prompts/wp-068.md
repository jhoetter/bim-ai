## Background
OSM-V3-02 is the render style for neighborhood masses: desaturated warm grey, no edge wires, suppressed in interior plan views. Visual rule: **neighbors must read as background, never as competing foreground.**

This WP is **frontend + CSS only**. No new data model. No new commands.

## Implementation tasks

### 1. Design token (`packages/web/src/tokens-drafting.css`)

Add to the `:root` block:
```css
--neighborhood-mass-color: #a8a39c; /* OSM-V3-02: neutral warm grey for context massing */
--neighborhood-mass-alpha: 0.85;
```

No hex literals elsewhere — all render code must reference `var(--neighborhood-mass-color)`.

### 2. Render path for `neighborhood_mass` elements

In the plan canvas rendering code, add a dedicated render case for `kind === 'neighborhood_mass'`:
- Footprint polygon filled with `var(--neighborhood-mass-color)` at `var(--neighborhood-mass-alpha)` opacity
- No stroke / edge wires
- Renders below all other elements (z-order: first)
- Suppressed when the active view is an interior plan (check `viewKind === 'floor_plan'` or equivalent; show in `site_plan`, `elevation`, `threed`)

### 3. Per-view toggle (store slice)

Add `showNeighborhoodMasses: boolean` (default `true`) to the view store or viewport settings slice. Expose a `toggleNeighborhoodMasses()` action. When false, neighborhood mass elements are excluded from the render pass entirely.

### 4. CSS check — zero hex literals

The component/render code must use `var(--neighborhood-mass-color)` only. A test should grep the changed files and assert no `#[0-9a-fA-F]{3,6}` hex literals appear in the new render case.

### 5. Tests (`packages/web/src/__tests__/neighborhoodMassRender.test.ts`)

Write at least 6 tests:
- `neighborhood_mass` element renders as a polygon (rendered output includes fill)
- Fill uses CSS var, not hex literal
- `showNeighborhoodMasses = false` → element excluded from render
- Interior plan view → neighborhood masses suppressed
- Site plan view → neighborhood masses visible
- Z-order: neighborhood masses render before walls (index check)
