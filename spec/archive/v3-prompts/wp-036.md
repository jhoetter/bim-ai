## Implementation tasks

### 1. Status-bar slot 6 entry (`packages/web/src/workspace/StatusBar.tsx`)

Find the `StatusBar.tsx` component. The status bar currently has the lens dropdown in slot 2 (CHR-V3-03). Add a slot 6 entry on the right end of the bar:

```tsx
<button
  className="status-bar__slot status-bar__activity-entry"
  aria-label="Activity stream"
  onClick={toggleActivityDrawer}
  title="Cmd+H"
>
  <ActivityIcon />
  {unreadCount > 0 && <span className="status-bar__badge">{unreadCount}</span>}
</button>
```

`unreadCount` = number of activity rows newer than the user's last-seen timestamp.

### 2. `ActivityDrawer` component (`packages/web/src/collab/ActivityDrawer.tsx`)

A right-edge drawer that slides in at 380 px width:

```tsx
type ActivityDrawerProps = {
  isOpen: boolean;
  onClose: () => void;
};
```

Behaviour:
- Slides in from the right edge in 200 ms using `--ease-paper` transition.
- Overlays the right rail (z-index above Inspector); right rail dims to 50 % opacity while drawer is open.
- Shows the latest 10 activity rows from `activityStore`.
- Each row at `--text-sm`: avatar + verb + noun + timestamp at `--text-2xs`.
- **Hover:** canvas dims to 40 % + previews the change (query the snapshot diff; show at `--cat-edit` overlay). Hover does NOT commit.
- **Click:** time-travel — sets the canvas to the row's `parentSnapshotId` with a "viewing version N" banner + floating `[Restore]` chip. Click `[Restore]` → `POST /activity/{id}/restore`.
- Filter chips at top: `All / Mine / Comments / Commits`.
- "See all →" link opens the full `ActivityPanel` (VER-V3-01).
- `Esc` or clicking outside closes.

### 3. `Cmd+H` keybinding (`packages/web/src/workspace/Workspace.tsx`)

```ts
useEffect(() => {
  const handler = (e: KeyboardEvent) => {
    if ((e.metaKey || e.ctrlKey) && e.key === 'h') {
      e.preventDefault();
      toggleActivityDrawer();
    }
  };
  window.addEventListener('keydown', handler);
  return () => window.removeEventListener('keydown', handler);
}, [toggleActivityDrawer]);
```

### 4. Drawer state (`packages/web/src/workspace/workspaceStore.ts`)

Add `isActivityDrawerOpen: boolean` and `toggleActivityDrawer()` to the workspace store (or create a minimal `activityDrawerStore.ts` if no workspace store exists). The store also tracks `lastSeenAt: number` (ms epoch) to compute `unreadCount`.
