  #### Asymmetric gable roof (KRN-11)

  ```

    kind: 'roof',
    roofGeometryMode: 'asymmetric_gable',
    footprintMm: [(0,0), (5000,0), (5000,8000), (0,8000)],
    referenceLevelId: '<First Floor>',

    eaveHeightLeftMm: 1500,                // west wall above upper floor: 1500mm tall (so absolute 4500mm)
    eaveHeightRightMm: 4000,               // east wall above upper floor: 4000mm tall (absolute 7000mm)

  ```
